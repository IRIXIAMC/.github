# IRIXIA MC

**Serveur Minecraft Skyblock — Irixia Forge**

> Le monde a été consumé par le Grand Effondrement. Reconstruis ton empire industriel dans le Vide Mécanique.

---

### Le concept

Skyblock Farm-to-Win avec un thème industriel unique. Machines custom, économie dual, items forgés, et un programme Farm-to-Freedom pour les meilleurs joueurs.

### Features

- Skyblock custom avec 15 rangs de progression + prestige
- Machines industrielles (Auto-Miner, Chunk Hopper, Ore Crusher, Forge Reactor...)
- Pioche du Vide évolutive (10 niveaux, soulbound)
- Économie dual : Bolts (farm) + IRIX (premium)
- Enchantements custom (20 enchants uniques)
- Pets utilitaires avec niveaux
- Donjons PvE instanciés ("Usines Corrompues")
- Caisses, Pass de Forge, Events saisonniers
- Auction House avec taxes dynamiques
- Programme Farm-to-Freedom (cashout gift cards)

### Stack

`Java 21` · `Paper 1.21.11` · `Velocity` · `Maven` · `PostgreSQL` · `Redis`
